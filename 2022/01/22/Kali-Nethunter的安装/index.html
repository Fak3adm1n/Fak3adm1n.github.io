2022.01.22 Fak3adm1n  热度 ℃

Kali Nethunter的安装

首先，你得去https://stats.nethunter.com/nethunter-kernels.html 看看有没有你的设备，如果没有，就要自己编译内核（麻烦的要死，建议不要）

没内核就只是识别不了网卡和不能HID，MSF之类的还是能正常使用的。

1. 有内核

你需要一个Linux系统的设备，理论上手机装Linux发行版也行。
然后安装git
apt install git
克隆地址
git clone https://gitlab.com/kalilinux/nethunter/build-scripts/kali-nethunter-project.git
构建准备

然后等，3gb多的大小，还好GitLab不慢
安装python2
apt install python2
还有pip

然后构建
python2 build.py -d cancrom -m -fs full
构建完成救行了
然后你要有TWRP
然后进REC，双清，刷上要求的系统(Mi4就是CM13)，重启，再进来，刷内核，就行了。
